
# Contribute 
## Installation 
```bash
    git clone git@github.com:Nicolas-Dmb/Wallet.git
    cd Wallet
```
    [intall uv](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer)
```bash
    uv venv
    uv pip install -r pyproject.toml
```

## Run project 
```bash
    python app/main.py
```
## Run tests
```bash
    uv run pytest tests
```
